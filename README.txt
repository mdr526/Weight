# پیگیری وزن — نسخهٔ PWA (قابل نصب)

## این پکیج چیست؟
این یک اپلیکیشن وب پیشرفته (PWA) است که می‌توانید آن را مثل یک اپ واقعی روی صفحهٔ اصلی
گوشی اندروید یا آیفون نصب کنید. بعد از نصب:
- آیکون مستقل روی صفحهٔ اصلی دارد (بدون نوار آدرس مرورگر)
- آفلاین هم تا حدی کار می‌کند (صفحهٔ اصلی برنامه کش می‌شود)
- ذخیرهٔ تصویر نمودار در گالری، اشتراک‌گذاری، و سایر قابلیت‌ها بسیار مطمئن‌تر از حالت
  پیش‌نمایش داخل چت کار می‌کنند، چون این نسخه دیگر داخل iframe ایزوله نیست.

## نکتهٔ مهم: نیاز به میزبانی HTTPS
سرویس‌ورکر (که نصب PWA و کارکرد آفلاین را ممکن می‌کند) فقط روی آدرس‌های HTTPS واقعی کار
می‌کند (نه با باز کردن مستقیم فایل از روی گوشی). باید این پوشه را روی یک میزبان وب رایگان
آپلود کنید. چند گزینهٔ رایگان و ساده:

### گزینه ۱: Netlify Drop (ساده‌ترین روش)
۱. به آدرس https://app.netlify.com/drop بروید
۲. کل این پوشه (همراه با زیرپوشهٔ icons) را در صفحه رها کنید (drag & drop)
۳. یک آدرس HTTPS رایگان فوراً ساخته می‌شود

### گزینه ۲: GitHub Pages
۱. این فایل‌ها را در یک ریپازیتوری گیت‌هاب آپلود کنید
۲. از تنظیمات ریپازیتوری، GitHub Pages را روی شاخهٔ اصلی فعال کنید
۳. بعد از چند دقیقه، آدرس HTTPS آماده می‌شود

### گزینه ۳: Vercel
۱. در vercel.com حساب بسازید و همین پوشه را به‌عنوان یک پروژهٔ استاتیک Deploy کنید

## نصب روی گوشی
بعد از اینکه آدرس HTTPS را در مرورگر گوشی باز کردید:

**اندروید (کروم):**
- منوی سه‌نقطه (⋮) → «افزودن به صفحهٔ اصلی» یا «نصب اپ»

**آیفون (سافاری):**
- دکمهٔ اشتراک‌گذاری (□↑) → «Add to Home Screen» / «افزودن به صفحهٔ اصلی»

## ساختار فایل‌ها
```
index.html          ← خود برنامه
manifest.json       ← اطلاعات نصب PWA (نام، آیکون‌ها، رنگ‌ها)
sw.js               ← Service Worker (کش و کارکرد آفلاین)
icons/
  icon-192.png            ← آیکون عادی (اندروید)
  icon-512.png            ← آیکون عادی بزرگ (اسپلش‌اسکرین)
  icon-maskable-192.png   ← آیکون با حاشیهٔ امن (اندروید adaptive icon)
  icon-maskable-512.png   ← نسخهٔ بزرگ همان
  apple-touch-icon.png    ← آیکون مخصوص iOS
  favicon-32.png          ← آیکون تب مرورگر
  favicon-16.png          ← آیکون تب مرورگر (ریز)
```

## دادهٔ شما کجا ذخیره می‌شود؟
تمام داده‌ها (وزن‌های ثبت‌شده) فقط در حافظهٔ محلی همان مرورگر/دستگاه ذخیره می‌شوند. اگر
اپ را پاک کنید یا کش مرورگر را خالی کنید، داده‌ها از بین می‌روند. این برنامه به هیچ سروری
داده ارسال نمی‌کند.
